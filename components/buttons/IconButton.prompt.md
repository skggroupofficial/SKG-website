**IconButton** — square, icon-only control. Always pass `label` for accessibility.

```jsx
<IconButton label="Menu"><i className="ph ph-list" /></IconButton>
<IconButton variant="outline" label="Search"><i className="ph ph-magnifying-glass" /></IconButton>
<IconButton variant="solid" label="Next"><i className="ph ph-arrow-right" /></IconButton>
```
